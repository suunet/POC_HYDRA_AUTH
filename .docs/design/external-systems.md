# external-systems — 外部システム定義

| ID | 外部システム | 説明 | スコープ |
|---|---|---|---|

| EXT-02 | Redis | レートリミット（ログイン失敗回数・ロック状態・パスワードリセット連続送信・メール確認再送信・招待トークン送信）の一時状態管理。TTLによる自動解除 | Must |
| EXT-03 | SMS/TOTPプロバイダー | 2段階認証のワンタイムパスワード送信 | Better |
