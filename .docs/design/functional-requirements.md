# functional-requirements — 機能要求定義

| ID | 機能要求 | 対応BUC | スコープ |
|---|---|---|---|
| FR-01 | メールアドレスとパスワードでユーザーを登録する機能（登録時に`user`ロール（VAR-08）をデフォルト付与。UX上はロールを意識させない。メールアドレス重複時も**201を返し**メール送信しない：ユーザー列挙攻撃対策（重複有無をレスポンスで区別しない）。レートリミット（VAR-16）を登録済み/未登録に関わらず一様に適用すること） | BUC-U01 | Must |
| FR-02 | メール確認トークンを送信する機能 | BUC-U01 | Must |
| FR-03 | メール確認トークンを検証しメール確認済みに更新する機能 | BUC-U02 | Must |
| FR-04 | メール確認トークンを再送信する機能（レートリミット（VAR-13）適用。未登録・確認済みメールアドレスの場合も200を返しメール送信しない：ユーザー列挙攻撃対策。再送信前に既存の有効トークンを無効化すること（CND-10）。メール送信失敗時は既存トークン無効化・新トークン保存を含むトランザクション全体をロールバックすること） | BUC-U03 | Must |
| FR-05 | メール/パスワードでJWTを発行する機能（発行時点のロールをClaimsに含めること） | BUC-U04 | Must |
| FR-06 | リフレッシュトークンでアクセストークンを再発行する機能（再発行時点のロールをClaimsに含めること） | BUC-U05 | Must |
| FR-07 | リフレッシュトークンを失効させる機能。自発的ログアウト（BUC-U06）はリクエストで送信されたRT1本のみを失効させ200を返す（`revocation_reason` 不要。自発的操作のため失効理由は自明）。冪等設計のため存在しない・既に失効済みのRTに対しても200を返す | BUC-U06 | Must |
| FR-08 | パスワードリセットトークンをメール送信する機能。登録済み・未登録に関わらず同一レスポンスを返すこと（ユーザー列挙攻撃対策） | BUC-U07 | Must |
| FR-09 | パスワードリセットトークンを検証しパスワードを更新する機能 | BUC-U07 | Must |
| FR-10 | 現在のパスワードを検証した上でパスワードを変更し全セッションを無効化する機能（レスポンスにパスワード変更による失効理由（VAR-10: `password_changed`）を含める） | BUC-U08 | Must |
| FR-11 | メールアドレスとロールを指定して招待レコード（メールアドレス・付与ロール・招待トークンのハッシュ）をDBに保存し、招待URLをメール送信する機能 | BUC-A01 | Must |
| FR-12 | 再招待時に既存の招待トークンを無効化し新トークンを発行する機能 | BUC-A01 | Must |
| FR-13 | 招待トークンを検証しパスワードを設定して管理者アカウントを有効化する機能（DBの招待レコードに紐付けられたロールをユーザーに付与すること） | BUC-A02 | Must |
| FR-14 | 指定したリフレッシュトークンを強制失効させる機能（レスポンスに強制失効理由（VAR-10: `forced_revocation`）を含める） | BUC-A03 | Must |
| FR-15 | 管理者アカウントを無効化しセッションを全失効させる機能（レスポンスに無効化による失効理由（VAR-10: `account_disabled`）を含める） | BUC-A04 | Must |
| FR-16 | 無効化済み管理者アカウントを再有効化する機能（再有効化後はセッションなし・手動ログイン必要） | BUC-A05 | Must |
| FR-17 | アカウントを論理削除しセッションを全失効させる機能（レスポンスに削除による失効理由（VAR-10: `account_deleted`）を含める）。自己削除時はパスワード再入力による本人確認を行うこと（CND-15） | BUC-A06 | Must |
| FR-18 | ロールを変更しセッションを全失効させる機能（レスポンスにロール変更による失効理由（VAR-10: `role_changed`）を含める） | BUC-A07 | Must |
| FR-19 | JWTを検証するミドルウェア機能 | — | Must |
| FR-20 | ヘルスチェックエンドポイント | BUC-S01 | Must |
| FR-21 | デバイス（セッション）一覧を返す機能 | BUC-U09 | Better |
| FR-22 | 指定デバイスのセッションを削除する機能 | BUC-U10 | Better |

> FR-19はユーザー起点のBUCではなくシステムが常に提供すべき機能のため対応BUCなし
